## 📝 Manual Editor
After filling in all sections, navigate to **Finalize & Export** (step 7).
Click **✏️ Manual Resume Editor** to expand the editor panel.
You can:
- Edit any personal detail, summary, experience, education, skills, projects, certifications, languages, achievements
- Add or remove entries with + / ✕ buttons
- Changes sync instantly to the resume data
- Click **Export PDF with Changes** to download with your edits
